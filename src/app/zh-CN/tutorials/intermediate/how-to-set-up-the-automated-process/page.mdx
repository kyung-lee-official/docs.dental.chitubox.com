# 如何设置自动化流程

自动化流程是 CHITUBOX Dental 的核心功能之一，可以帮助齿科专业人员简化工作流程、提高工作效率，但在使用时需要注意一些细节。
在当前的版本中，自动化流程主要支持五种应用场景，分别是正畸牙模、修复牙模、种植导板、磨牙套、临时冠。不同应用的自动流程内置的算法及参数不同，一定要选择对应的应用场景进行自动处理。如果想要处理种植体导板，但选择的是正畸牙模的应用场景，模型的定向逻辑和处理流程不正确，就会导致结果不理想。
第一次使用软件，在导入模型前需要先选择打印机及应用，同时可以在“设置”中更改自动流程的步骤和调整功能参数。此外还可以在”高级设置“中修改材料参数和打印参数，并且根据自身打印需求开启抗锯齿、缩放补偿等精度补偿功能，用以调整屏幕透光度，提高模型的精确度。

完成设置进入操作界面之后，用户只需要导入模型，然后点击页面上方的“自动处理”按钮，即可一键完成所有模型处理步骤，最后点击“切片”即可开启打印流程。

若是用户想要修改设置，也可以在左下角的项目名称处，点击名称旁边的箭头从“配置设置”选项进入设置页，调整完成后点击“应用”即可。

如果需要经常处理不同的应用类型，用户可以预先创建好不同的配置，配置会自动保存在软件中，下次打开软件时直接在对应的应用场景中处理模型即可。为了方便区分，还可以修改配置名称，这样每次打印时可以迅速运用对应场景的自动流程，大大地提高工作效率和便捷性。
