# 设置支撑参数

点击主窗口左侧的支撑图标 <ChituboxSupport /> 即可打开 <Link href="/zh-CN/user-manual/latest/ui-and-features/operations#支撑">支撑面板</Link>。

CHITUBOX 提供了细、中、粗三种默认支撑设置可选。如果它们不适合您的模型，您可以点击 <ChituboxProfileAddIcon />“添加配置” 按钮，添加新的支撑配置文件并自定义支撑设置。 还可以在这里 <ChituboxProfileRenameIcon /> 重命名、 <ChituboxProfileSettingsIcon /> 编辑和 <ChituboxProfileDeleteIcon /> 删除支撑设置。

## 支撑设置

### 常规

支撑的常规设置。

| 参数                       | 说明                                                         |
| -------------------------- | ------------------------------------------------------------ |
| 模型抬升高度               | 模型最低点与打印平台之间的高度距离，默认值为 5 mm。          |
| 自动支撑允许最大三角面角度 | 允许自动支撑点所在的模型表面（三角面）与平台之间的最大角度。 |
| 自动支撑点间距             | 自动支撑时，支撑点之间的距离。                               |
| 自动支撑主干间距           | 自动支撑时，支撑主干之间的距离。                             |

### 上段

-   支撑上段尖端

| 参数           | 说明                               |
| -------------- | ---------------------------------- |
| 与模型接触形状 | 支撑与模型接触的形状。             |
| 形状直径       | 支撑接触点为球形时，接触点的直径。 |
| 顶部接触深度   | 支撑与模型接触的深度。             |

-   支撑上段主体

| 参数                    | 说明                        |
| ----------------------- | --------------------------- |
| 连接杆截面形状          | 支撑上段连接杆的截面形状。  |
| 连接杆上端直径          | 支撑上段连接杆上端的直径。  |
| 连接杆下端直径          | 支撑上段连接杆下端的直径。  |
| 连接杆长度              | 连接杆的长度。              |
| 连接杆与 Z 方向最大夹角 | 连接杆与 Z 方向的最大夹角。 |

### 主干

| 参数         | 说明                 |
| ------------ | -------------------- |
| 主干截面形状 | 支撑主干截面的形状。 |
| 主干上端直径 | 支撑主干上端的直径。 |
| 主干下端直径 | 支撑主干下端的直径。 |

### 底段

| 参数           | 说明                       |
| -------------- | -------------------------- |
| 下端直径       | 支撑底段的下端直径。       |
| 底段高度       | 支撑低段的高度。           |
| 与平台接触形状 | 支撑与平台接触的形状。     |
| 形状直径       | 支撑与平台接触形状的直径。 |
| 形状厚度       | 支撑与平台接触形状的厚度。 |

### 底阀

| 参数             | 说明                                 |
| ---------------- | ------------------------------------ |
| 底阀类型         | 底阀的形状。                         |
| 底阀面积比例     | 底阀相对模型水平投影面积的比例大小。 |
| 底阀厚度         | 底阀的厚度。                         |
| 滑板边缘凸起高度 | 底阀边缘凸起的高度。                 |
| 网口边长         | 网口的边长。                         |
| 网口宽度         | 网口的宽度。                         |

### 小柱子

小柱子相较于常规支撑更细更小，用来支撑模型结构较细的部分。

| 参数             | 说明                       |
| ---------------- | -------------------------- |
| 直径             | 小柱子直径。               |
| 最大长度         | 小柱子的最大长度。         |
| 与模型接触点形状 | 小柱子与模型接触点的形状。 |
| 接触深度         | 小柱子接触模型的深度。     |

### 交叉结构

交叉结构用于连接不同支撑以增加支撑整体强度。

| 参数                 | 说明                                             |
| -------------------- | ------------------------------------------------ |
| 启用交叉结构         | 自动支撑时，在支撑主干之间自动生成交叉结构。     |
| 直径                 | 交叉结构的直径。                                 |
| 起始高度（相对平台） | 生成交叉结构的起始高度（相对于平台）。           |
| 最小间距（Z）        | 两组交叉结构在 Z 方向上的间距。                  |
| 最大间距（XY）       | 生成交叉结构时，两个支撑主干之间的最大间距。     |
| 双向生成交叉结构     | 手动添加支撑时，识别两侧支撑并生成双向交叉结构。 |

### 高级

支撑的高级参数设置。

| 参数                             | 说明                                                                                      |
| -------------------------------- | ----------------------------------------------------------------------------------------- |
| 模型与支撑主干的最小间距         | 模型主体与支撑主干的最小间距。                                                            |
| 支撑主干与接触点的最大水平偏移量 | 支撑主干与模型接触点在 XY 水平面的最大允许水平偏移量。                                    |
| 自动支撑规避圆孔结构             | 开启后，使用自动支撑会避开圆孔结构。                                                      |
| 增强自动支撑                     | 启用/关闭增强自动支撑。共有三种增强模式：独立强化（IR）、三角强化（TR）和全局强化（GR）。 |

## 添加支撑

### 自动添加

点击“自动支撑”按钮，将按照指定支撑配置为所选模型自动添加支撑。

### 手动添加

光标移动到所需位置并点击鼠标左键以添加支撑。

## 删除支撑

点选或框选想要删除的支撑，按下 Delete 键或删除键删除选中的支撑。

也可以点击“移除全部支撑”按钮，一键移除选中模型所加的全部支撑。
