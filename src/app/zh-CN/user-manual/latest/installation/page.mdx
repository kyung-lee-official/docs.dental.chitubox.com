# 安装

## 系统要求

<SystemRequirements lang={"zh"} />

## Windows

从我们的 <Link href="https://dental.chitubox.com/zh-Hans/download">官网</Link> 下载安装包，双击下载的安装程序开始安装过程。

<ImageContainer
	src={
		"/images/docs/zh-CN/user-manual/1.x.x/002-installation-windows-step1.png"
	}
/>

点击 "下一步" 继续。

<ImageContainer
	src={
		"/images/docs/zh-CN/user-manual/1.x.x/002-installation-windows-step2.png"
	}
/>

阅读最终用户许可协议，如果您同意条款，请勾选 "我接受许可协议" 选项，然后点击 "下一步" 继续。

<ImageContainer
	src={
		"/images/docs/zh-CN/user-manual/1.x.x/002-installation-windows-step3.png"
	}
/>

选择安装路径，我们建议您使用默认路径，然后点击 "下一步" 开始安装过程。

<ImageContainer
	src={
		"/images/docs/zh-CN/user-manual/1.x.x/002-installation-windows-step4.png"
	}
/>

等待安装完成。

## macOS

在 macOS 上使用 CHITUBOX Dental 无需安装流程，因为它是以可转移 (portable) 的形式提供的。您只需下载 CHITUBOX Dental 的最新版本，然后将其拖放到“应用程序”文件夹中即可。
