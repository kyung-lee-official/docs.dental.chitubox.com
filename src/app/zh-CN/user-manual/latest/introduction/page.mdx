# 介绍

CHITUBOX Dental 是一款面向齿科的 3D 打印切片软件，可以将 3D 网格文件切片并转换为可打印文件。 它旨在减少用户上手成本，简化 3D 打印过程，让 3D 打印更轻松、高效，同时它还提供了广泛的功能和定制选项，可以大幅提高打印质量并降低打印失败的风险。
