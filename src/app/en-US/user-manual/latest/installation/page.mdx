# Installation

## System Requirements

<SystemRequirements />

## Windows

Download the installation package from our <Link href="https://dental.chitubox.com/en/download">official site</Link> and double-click the downloaded installer to start the installation process.

<ImageContainer
	src={
		"/images/docs/en-US/user-manual/1.x.x/002-installation-windows-step1.png"
	}
/>

Click "Next" to continue.

<ImageContainer
	src={
		"/images/docs/en-US/user-manual/1.x.x/002-installation-windows-step2.png"
	}
/>

Read the EULA, check the "I accept the license" option if you agree to the terms, and click "Next" to continue.

<ImageContainer
	src={
		"/images/docs/en-US/user-manual/1.x.x/002-installation-windows-step3.png"
	}
/>

Select the installation path, we recommend you to use the default path, and click "Next" to continue.

<ImageContainer
	src={
		"/images/docs/en-US/user-manual/1.x.x/002-installation-windows-step4.png"
	}
/>

Wait for the installation to complete.

## macOS

CHITUBOX Basic for macOS doesn't requires an installation process since it is a portable application. You can simply drag the CHITUBOX icon to the Applications folder to install it.
