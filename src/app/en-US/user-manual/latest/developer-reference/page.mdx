# Developer Reference

## Invode CHITUBOX Dental with Mesh Files from Command Line

### Windows

`executable file [file ...]`

Example:

```bash
'C:\Program Files\CHITUBOX Dental' 'C:\Users\user\Documents\3D Models\test1.stl' 'C:\Users\user\Documents\3D Models\test2.stl'
```
